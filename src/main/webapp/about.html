Contract

After merge make a new branch!
We agree to:

-Push if tests are green
-If you make a method, make a test for it
-Push often
-Write in discord when you push
-Commit messages should be in english
-Use this format for commit messages: “What have you changed, How and why (if relevant)”
-Remember NOT to push anything that won’t build, and don’t push before pulling
(git add . git commit mvn clean test => if green git push)
-Check in daily with status on work progress
-Do the work agreed upon
-Show up for group meetings/class (unless there’s a case of illness or other unfortunate events)
-If stuck on a problem, ask for help.
-Check messages in a timely manner Repeated breach of contract will lead to expulsion of the group.
